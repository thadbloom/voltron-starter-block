# ⚡️ Voltron Starter Block ⚡️
Experimental starter block scaffolded with Create Block tool.

## Install

```
npm install
```

## Develop

### Start local wp instance (requires Docker):
```
npm run wp-env start
```

### Create block build process:
```
npm start
```

## Deploy

### Creates production build
```
npm build
```
